## Vergleichsoperatoren
| Operator | Beschreibung                                    |
|----------|-------------------------------------------------|
| $eq      | Gleichheit                                      |
| $ne      | Ungleichheit                                    |
| $gt      | Größer als                                      |
| $gte     | Größer als oder gleich                          |
| $lt      | Kleiner als                                     |
| $lte     | Kleiner als oder gleich                         |
| $in      | Übereinstimmung mit einer beliebigen Werteliste |
| $nin     | Keine Übereinstimmung mit einer Werteliste      |
